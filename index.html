Pet Projects
BadeBhai lang is my ongoing project which is a new interpreted programming language being developed as a succesor to Bhai lang with added funcionalities including garbage collection. This project is entirely made in Rust.
MOTOP or as how I like to call it Making Of The Optimus Prime was my first project into research based learning. It is auto regressive decoder-only transformer based LLM with 124M parameters that I made and trained from scratch along with a similar mamba based model.
Checking Tree isomerism. It began as my attempt to solve graph isomerism problem, which remains unsolved till this day. (I am huge fan and follower of graph theory). It is a application of the AHU algorithm in C++. I also developed an interactive web platform to visually create the graphs and generate their adjacency lists. It is also the project that sparked the inquisitiveness in me for algorithms and graphs.
